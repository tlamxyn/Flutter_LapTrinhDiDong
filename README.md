## Hướng dẫn tải và chạy đồ án

> 1. Mở ``VisuoStudioCode``
> 2. Mở ``Terminal``
> 3. Nhập lệnh ``git clone https://github.com/tlamxyn/Flutter_Laptrinhdidong.git``

> 5. Chọn thiết bị chạy
